# 📊 Data Cleaning & Visualization Project

## 📌 Overview
This project demonstrates the process of cleaning, preprocessing, and visualizing a raw dataset using Python. It includes handling missing values, removing duplicates, treating outliers, and creating meaningful visualizations to gain insights from the data.

---

## 🎯 Objectives

- Clean and preprocess raw data
- Handle missing values, duplicates, and outliers
- Perform Exploratory Data Analysis (EDA)
- Visualize important insights using graphs
- Generate a cleaned dataset for further analysis

---

## ✨ Features

- ✅ Data Cleaning
- ✅ Missing Value Handling
- ✅ Duplicate Removal
- ✅ Outlier Detection
- ✅ Data Visualization
- ✅ Cleaned Dataset Export

---

## 🛠️ Technologies Used

- Python
- Pandas

- Matplotlib

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/rajeshparida96688-ops/Data-Cleaning-Visualization.git
```

Move into the project folder:

```bash
cd Data-Cleaning-Visualization
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

---

## ▶️ Run the Project

Open Jupyter Notebook:

```bash
jupyter notebook project.ipynb
```

---

## 📊 Visualizations Included

- Histogram

- Scatter Plot
- Correlation Heatmap
- Bar Chart
- Distribution Plot

## 📈 Project Workflow

1. Load the dataset (`data.csv`)
2. Explore the data
3. Handle missing values
4. Remove duplicate records
5. Detect and treat outliers
6. Visualize the cleaned data
7. Save the cleaned dataset (`cleaned_data.csv`)
